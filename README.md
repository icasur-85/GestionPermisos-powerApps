# TFG – Gestión de permisos de usuarios en entornos Microsoft 365

Este repositorio contiene el código fuente del proyecto desarrollado como parte del
**Trabajo de Fin de Estudios del Grado en Ingeniería Informática de UNIR**.

## Descripción del proyecto
Aplicación desarrollada con Power Apps para la gestión y consulta de permisos y concesiones
utilizando como repositorio de datos listas de SharePoint Online. La aplicación incorpora
lógica de filtrado dinámico, automatización de procesos y exportación de datos a Excel
mediante flujos de Power Automate.

## Tecnologías utilizadas
- Microsoft Power Apps (Power Fx)
- Microsoft Power Automate
- SharePoint Online (listas)
- Excel Online (Business)
